# Random Quote Generator

**Автор:** Роман Малин

## Описание
приложение для генерации случайных цитат с сохранением истории.

## Функции
- Генерация случайных цитат
- История всех сгенерированных цитат
- Фильтрация по автору и теме
- Добавление новых цитат
- Сохранение истории в JSON
- Валидация ввода

## Установка и запуск
```bash
# Клонирование репозитория
git clone https://github.com/ваш-username/random-quote-generator.git
cd random-quote-generator

# Запуск (требуется Python 3.6+)
python quote_generator.py
